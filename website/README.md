# .ontos Protocol Website

This directory contains the public documentation and launch site source.

Build locally:

```bash
npm run docs:build
```

The build output is written to `website/dist` and can be deployed to any static
host.

Current sections:

- home
- quickstart
- hosted viewer demo
- examples gallery
- AI context pack demo
- local-first workflow
- launch assets
- docs index

Launch assets:

- `website/src/assets/viewer-screenshot.svg`
- `website/src/assets/cli-screenshot.svg`
- `website/src/assets/social-preview.svg`
